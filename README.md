The implementation code for IEEE TSP paper "Semantic-based Distributed Learning for Diverse and Discriminative Representations".

In ./data, CategoryToClients10/LocalDist_iid_test10/LocalDist_niid10 correspond to i.i.d condition in Section VI-A with 10 nodes;

CategoryToClients4/LocalDist_niid_test4/LocalDist_niid4 correspond to the first non-i.i.d. condition in Section VI-B with 4 nodes;

CategoryToClients5_4/LocalDist_niid_test5_4/LocalDist_niid5_4 correspond to the second non-i.i.d. condition in Section VI-B with 5 nodes.
